Formula mock prezzo (priceCalculator.ts)

// PCB base cost
pcbBase = (width_mm × height_mm / 1000) × layerMultiplier × materialMultiplier
layerMultiplier: { 1:1.0, 2:1.2, 4:2.1, 6:3.5, 8:5.0 }
materialMultiplier: { FR4:1.0, Flex:2.5, Aluminum:1.8 }

// Engineering fee fisso
engineeringFee = 3.38

// Qty scaling (economie di scala)
qtyFactor = qty ≤ 5 ? 1.0 : qty ≤ 10 ? 0.85 : qty ≤ 50 ? 0.7 : 0.55

// PCBA (se assembly attivo)
setupFee = type==='Economic' ? 6.76 : 15.00
stencilFee = side==='Both' ? 2.54 : 1.27
componentFee = bomItems.filter(dnp=false).length × 0.08
smtFee = cplItems.length × 0.004

// Totale
total = (pcbBase + engineeringFee) × qtyFactor + setupFee + stencilFee + componentFee + smtFee

  body: { pcbConfig, assemblyConfig, bomItems }
  → { breakdown: PriceBreakdown, buildTime: string }

  body: OrderState (senza File binari)
  → { id: string, pdfUrl: string }

POST /api/configuratore/richiedi-ordine
  body: { configId, contatto: { nome, email, azienda } }
  → { success: boolean, ticketId: string }

Error handling & UX

ZIP gerber non valido
Toast errore + highlight zona drop. Lista file rilevati mostrata. Richiede ≥1 .GTL.
BOM/CPL formato errato
Preview prime 3 righe + messaggio "colonna X non trovata". Suggerimento formato atteso.
Componenti DNP
Badge giallo "DNP" su riga. Non contati nel prezzo assembly. Tooltip spiegazione.
Draft auto-save
localStorage ogni cambio di step. Banner "Bozza ripristinata" al reload. Pulsante "Nuova configurazione" per reset.
